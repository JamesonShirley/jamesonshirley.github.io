Court Case Management System

A court case management system built with Python Django and JavaScript next.js. It allows for courts to schedule hearings and keep track of them in a calendar. It also allows for the creation of cases and the tracking of case information, as well as court orders and other documents. It keeps track of the parties involved in the case and the attorneys representing them.

Grocery Express

This is a java application to monitor fictional drones delivering groceries. It has the ability to calculate distances between deliveries and ensures the drones have enough battery to make the deliveries. It does batch jobs of the drones every night and ensure everything is running smoothly.

Scheduling Application

This application was built in java using FXML. It manages customers and appointments for a fictional business. It shows a calendar of all current appointments, and allows for the creation of new appointments and customers. It also has a report that shows the number of appointments by type and month.
